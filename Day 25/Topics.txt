Topics: 

- Working with CSV files

- Analysing data with Pandas


Project: U.S. States Game (using Turtle and Pandas). https://www.sporcle.com/games/g/states

Breakdown the Problem:

1. Convert the guess to Title case.

2. Check if the guess is among the 50 states.

3. Write the correct guess onto the map.

4. Use a loop to allow the user keep guessing.

5. Record the correct guesses in a list

6. Keep track of the score.

7. Generate CSV file of missed states.

Project: Quiz Game

Steps:
1. Create a model for a Question class
   - Attributes must be created when new Question object is initialized.

2. Create a question bank of Question objects.

3. Ask user to answer a question.
   - Ask question.
   - Check if answer is correct.
   - Check if end of quiz.
   
4. Tell the user the final score when the questions are finished.

5. Get more questions and answers from the Open Trivia Database (OTB) API (https://opentdb.com/api_config.php).
   - Generates questions and answers in JSON format (similar to Python dictionaries).
